# Relations

Components for displaying and editing relations between resources.

- **`RelationCell`** — renders a related entity inside a table cell.
- **`DisplayInline`** — inline display of a related entity.
- **`RelationControlRenderer`** — JSON Forms renderer for editing a relation in a form.
- **`RelationReadonlyRenderer`** — read-only variant.
- **`RelationModal`** — pick or edit a related entity in a modal.
- **`useRelationBinding`** — composable wiring a relation control to the API.
